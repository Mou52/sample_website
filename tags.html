Important HTML Tags

This page is a cheat-sheet for HTML tags. It will also use tables with table and other things.

The tables don't have every singular tag that is usable in HTML, but they are the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data (like so)
th table header include a table header
td table data include a table data cell
